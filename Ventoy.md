# [Ventoy](https://www.ventoy.net)

### Download Ventoy
```bash
wget https://github.com/ventoy/Ventoy/releases/download/v1.0.94/ventoy-1.0.94-linux.tar.gz
tar -xvf ventoy-1.0.94-linux.tar.gz
```

### Find Ventoy partition
```bash
lsblk
```

### Upgrade Ventoy
```bash
sudo ./Ventoy2Disk.sh /dev/sda -u
```
